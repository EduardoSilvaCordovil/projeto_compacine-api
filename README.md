## Instalando as depedências:
Abra o terminal e navegue até a pasta do seu projeto. Depois, digite:

```bash
npm install

npm install express dotenv mongoose

npm install nodemon
```
Para iniciar a aplicação
```bash
nodemon src/app.js
```
